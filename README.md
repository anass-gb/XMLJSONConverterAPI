# XMLJSONConverterAPI ⚡

Une application JavaFX pour **convertir facilement du XML en JSON et vice-versa**.  
Cette version utilise **l'API org.json** pour une conversion fiable et rapide.

---

## Fonctionnalités

- Conversion **XML → JSON** et **JSON → XML**
- Interface **JavaFX moderne et réactive**
- Chargement et sauvegarde de fichiers XML/JSON
- Zone d’entrée et de sortie avec aperçu en temps réel
- Support des attributs et des structures imbriquées
